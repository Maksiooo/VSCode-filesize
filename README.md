# VSCode File Size

Displays file size in the status bar.

<img src="https://daydottime.tk/vsc-extension-1.gif">

# Options

- Position (left or right)

- Priority (Higher values more to the left)

- Hide/Show status bar (use crtl + shift + p and search)
